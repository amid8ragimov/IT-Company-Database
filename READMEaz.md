# IT Company Database (SQL Layihəsi)

Bu layihə IT şirkətinin verilənlər bazasını simulyasiya edir.  
Layihədə işçilər, departamentlər, layihələr və onların əlaqələri modelləşdirilmişdir.

## Məqsəd

Bu layihənin əsas məqsədi SQL biliklərini nümayiş etdirməkdir:
- Verilənlər bazasının dizaynı
- Cədvəllər arasında əlaqələr (FOREIGN KEY)
- Many-to-many əlaqələr
- Analitik SQL sorğuları
- Aqreqat funksiyalar (COUNT, SUM, AVG)
- JOIN əməliyyatları

## Verilənlər Bazasının Strukturu

Layihə aşağıdakı cədvəllərdən ibarətdir:

- **Departments** — şirkətin departamentləri  
- **Employees** — işçilər haqqında məlumat  
- **Projects** — layihələr  
- **Employee_in_Project** — işçilərin layihələrdə iştirakı  

## Xüsusiyyətlər

- Realistik məlumatlar (işçilər, maaşlar, layihələr)
- Cədvəllər arasında düzgün əlaqələr
- Many-to-many münasibətlərin tətbiqi
- Analitik sorğular vasitəsilə məlumatların təhlili
- Təmiz və strukturlaşdırılmış SQL kod

## Nümunə Sorğular

- Hər departament üzrə işçi sayı  
- Ən yüksək maaş alan işçilər  
- Layihələr üzrə işlənmiş saatlar  
- Ən bahalı layihələr  
- İşçilərin layihələrdə rolları
